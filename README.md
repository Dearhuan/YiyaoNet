项目目录说明：

YiyaoNet：项目仓库

Yiyao：项目文件夹

alipay：支付宝沙箱环境，支付页面，回调页面

api：项目数据接口文件夹

css：CSS样式文件夹

doc：文档文件夹

html：HTML页面文件夹

iconfont：字体图标文件夹

images：图片文件夹

js：JavaScript文件夹

json：JSON数据文件夹

sql：数据库导出的SQL文件夹
